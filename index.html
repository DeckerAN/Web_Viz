Weather vs. Latitude

The purpose of this project was to analyze how weather changes based on a city's distance from the equater. To accomplish this analysis, we first pulled data from the OpenWeatherMap API on over 500 random cities from around the globe.


After assembling this dataset, we used the Matplotlib library to plot various measurable aspects of weather in these cities against their geographic latitude. Considered factors included temperature, cloudiness, wind speed, and humidity. This site provides the source data as well as the visualizations created to identify any trends. Each visualization is accompanied by a description of any discernable trends.
